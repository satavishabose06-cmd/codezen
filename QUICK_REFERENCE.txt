🐛 IF SOMETHING GOES WRONG:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Problem: "Connection refused"
  ─────────────────────────────
  → Make sure Django terminal shows "Starting development server"
  → Start Django BEFORE Streamlit

  Problem: "Cannot connect to backend"
  ─────────────────────────────────────
  → Django not running in Terminal 1?
  → Check terminal for errors
  → Restart both

  Problem: Any other error
  ────────────────────────
  → Run: verify.bat
  → Check terminal output
  → Read COMPLETE_SETUP.md

  Need database reset?
  ──────────────────
  → cd credit_app/creditproject
  → del db.sqlite3
  → python manage.py migrate
