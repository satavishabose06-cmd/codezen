## 🔧 Troubleshooting

### Issue: "Connection refused"
**Solution**: Make sure Django is running FIRST, then Streamlit

### Issue: "Module not found"
**Solution**: Run `pip install -r requirements.txt --upgrade`

### Issue: "Model not found"
**Solution**: Model files exist at `credit_app/ml_model/model.pkl` ✓

### Issue: Port already in use
**Solution**: Use different port in run scripts

### Issue: Any other error
**Solution**: Run `verify.bat` to diagnose
